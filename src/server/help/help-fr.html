Fonctionnement du calendrier

Le calendrier repliable de l'application permet de filtrer les dates.
  • Pour sélectionner une journée : cliquer sur la date.
  • Pour sélectionner une plage de durée : cliquer sur le premier jour de la plage et cliquer droit ou Ctrl-cliquer sur le dernier jour.
  • Le début et la fin de la plage de durée peuvent être sur des mois différents.
  • Pour tout désélectionner : double-cliquer sur une date.
  • Pour sélectionner un mois entier, cliquer sur le nom du mois.
  • Pour désélectionner un mois entier, double-cliquer sur le nom du mois.
Une fois le calendrier replié, la date (ou la plage de dates) sélectionnée est affichée dans le titre.
